To use [drawer navigator](https://reactnavigation.org/docs/drawer-based-navigation) you'll need to install some extra dependencies.

<Terminal
  cmd={[
    '$ npx expo install @react-navigation/drawer react-native-gesture-handler react-native-reanimated',
  ]}
/>

Next, you'll need to update your **babel.config.js** to include the reanimated plugin:

{/* prettier-ignore */}
```js babel.config.js
module.exports = {
  presets: [
      /* @hide ... */ /* @end */
    ],
    plugins: [
      /* @hide ... */ /* @end */
      'react-native-reanimated/plugin',
    ],
};
```

> After changing your `babel.config.js`, once run `npx expo start --clear` to start the development server and clear the babel cache.

<Tabs>
<Tab label="SDK 49 and lower">

Now you can use the `Drawer` layout to create a drawer navigator.

```jsx app/_layout.js
import { Drawer } from 'expo-router/drawer';

export default function Layout() {
  return <Drawer />;
}
```

</Tab>

<Tab label="SDK 50 and greater">

Now you can use the `Drawer` layout to create a drawer navigator. You'll also need to wrap the `<Drawer />` in a `<GestureHandlerRootView />` to enable gestures, this is required as of Expo Router v3 and greater. You only need one `<GestureHandlerRootView />` in your component tree, so any nested routes will not need to be wrapped.

```jsx app/_layout.js
import { GestureHandlerRootView } from 'react-native-gesture-handler';
import { Drawer } from 'expo-router/drawer';

export default function Layout() {
  return (
    <GestureHandlerRootView style={{ flex: 1 }}>
      <Drawer />
    </GestureHandlerRootView>
  );
}
```

Be careful about using `react-native-gesture-handler` on web as it can increase the JavaScript bundle size significantly. Learn more in [platform-specific modules](/router/advanced/platform-specific-modules/).

</Tab>
